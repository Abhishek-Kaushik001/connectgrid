# ConnectGrid 🚀

A full-stack social media web application built using MERN stack.

## 🔧 Tech Stack

- MongoDB
- Express.js
- React.js
- Node.js

## ✨ Features

- User Authentication (JWT)
- Create & Delete Posts
- Like System
- Responsive UI

## 📁 Folder Structure

- frontend/
- backend/

## ▶️ Run Locally

### Backend

cd backend
npm install
npm start

### Frontend

cd frontend
npm install
npm start
